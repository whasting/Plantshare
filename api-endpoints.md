# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Plants

- `GET /api/plants`
- `POST /api/plants`
- `GET /api/plants/:id`
- `PATCH /api/plants/:id`
- `DELETE /api/plants/:id`

### Tags

- `GET /api/tags`
- `POST /api/tags`
- `DELETE /api/tags/:id`
